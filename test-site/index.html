FAQ

Will the extension translate visible content first?

It should. Scroll slowly and watch the top sections update before the bottom of the article.

Will repeated text be cached?

Yes, repeated strings like menu labels and callouts should reuse previously translated text.

Will the progress bar move in real time?

It should update per batch. If it feels stuck, the batch size may be too large.
